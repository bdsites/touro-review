Intro to HTML

In this week's class we learn the background of the Web and web programming languages. We understand how to navigate the computer via the Terminal and install and use Atom.

We learn how to create new html files and their basic elements and saw how boring they look when you're just using html alone (like this page!). We can add meta data to our files, comment out lines, and add headings and paragraphs.

We learned how to make handy lists like this:

  1. One is Hashem!
  2. Two are the Luchos that Moshe brought
  3. Three are the Fathers

And like this:

  • Four are the Mothers
  • Five are the Books of the Torah
  • Six are the Books of the Mishnah

And we can do all kinds of neat lists like these:

  • Seven are the days of the week
  • Eight are the days til the Bris
  • Nine are the months until a baby's born

By the end of this class, you should be able to:

  1. Create a new file in Terminal
  2. Create a new folder in Terminal
  3. Delete a file in Terminal
  4. Delete a folder in Terminal
  5. Check your current directory
  6. See list of directories within your current location
  7. Change current directory
  8. See previous history
  9. Open a file
  10. Create new html document
  11. Navigate within Atom
  12. Add title to html file
  13. Create headings, paragraphs, and lists in an html file
  14. Comment out lines in html
