Intro to CSS

In this class, we learn how to see the backend of sites using developer tools. We begin an introduction to CSS, understanding properties and selectors, and the formatting potential that's within CSS. We learn the three ways to define a color, as well as the priority given when there are conflicting CSS commands. We also learn three ways to bring CSS into an HTML file.

We also learn how to make lists appear horizonally, like a nav bar:

  • HOME
  • ABOUT ME
  • CONTACT

By the end of this class, you should be able to:

  1. Access Chrome Developer Tools
  2. Add CSS into an HTML file in three ways
  3. Create a horizontal list
  4. Add a background image to a page
  5. Define a color in three ways
  6. Add opacity to a color
  7. Understand the meaning of "cascading" within CSS
  8. Add basic formatting to an html element, such as color, size, border
