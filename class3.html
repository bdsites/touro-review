Intro to CSS

In this class, we learn how to see the backend of sites using developer tools. We begin an introduction to CSS, including properties and selectors and the various options in formatting HTML elements. We also learn three different ways to define a color within CSS and the priority given when there are conflicting CSS commands. The three ways we learn to bring CSS into an HTML file are:

  1. Link to an external stylesheet in the <head> of the document
  2. Inline CSS
  3. Include the CSS code in the <head> of file

      4. We also learned how to make lists appear horizonally, like a nav bar:

      • HOME
      • ABOUT ME
      • CONTACT

      By the end of this class, you should be able to:

      1. Access Chrome Developer Tools
      2. Add CSS into an HTML file in three ways
      4. Create a horizontal list
      6. Add a background image to a page
      7. Define a color in three ways
      8. Add opacity to a color
      9. Understand the meaning of "cascading" within CSS
